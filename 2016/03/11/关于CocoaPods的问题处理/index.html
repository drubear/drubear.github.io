关于CocoaPods的问题处理

  • iOS

1.CocoaPods安装问题

文／姚宁（简书作者）
原文链接：http://www.jianshu.com/p/df61ea673838

原因：淘宝停止基于 HTTP 协议的镜像服务 需要改用https的协议fix步骤：

gem sources –remove http://ruby.taobao.org/

gem sources -a https://ruby.taobao.org/

gem sources -l

sudo gem install cocoapods

2.Pod Install问题

1.找不到github.com
2.持续Updating local specs repositories

可选择暂时先不更新本地库。找时间单独执行
pod repo update

从pod install改为
pod install –verbose –no-repo-update
