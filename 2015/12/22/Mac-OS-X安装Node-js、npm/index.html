Mac OS X安装Node.js、npm

转载请注明原作者，如果你觉得这篇文章对你有帮助或启发，不用请我喝咖啡:D

一、先安装Node.js

  1. 下载node.js的pkg。
  2. 按照官网上编写并保存example.js。
  3. 打开终端 路径浏览到example.js所在位置。
  4. 可以用官网上的简单例子跑hello world服务的那个，输入命令 node看看有没有安装成功。

二、查看npm是否正常工作**

  1. 终端输入npm -v能得到版本号，表示npm已正常工作，可以进行后续的npm install了。

    三、用npm安装需要的模块

    好了，由于网络强国的原因，官方的npm安装可能被墙或者慢。
    我们使用国内的一些镜像替换官方的

    https://registry.npmjs.org

1. 终端输入

npm config set registry http://registry.npm.taobao.org

2. 设置代理

如果你的网络连接使用了代理服务器，为npm设置代理，输入

npm config set proxy http://172.16.10.7:80

以及

npm config set http-proxy http://172.16.10.7:80

P.S. 1:不确定是哪一条在工作或者两条都有才能工作，总之这样设置后连接成功了。除了proxy、http-proxy，还有个值是https-proxy，失败的话可以多试试。

P.S. 2:需在代理服务器IP前加上http:// 或者https://，如果有用户名的话，这样设置：

http://username:password@server:port

3.执行安装

开始安装，这次npm install -g strong-cli后有了200响应，但我得到了一个错误：

猜测是代理设置错了。
把npm config set proxy 10.1.x.x:xxx 换成：

npm config set proxy http://10.1.x.x:xxx

之后（同样对http-proxy也做同样的改动），再次npm install 有反馈了。
